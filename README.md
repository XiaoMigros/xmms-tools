# XiaoMigros MuseScore Tools

### [Download](https://github.com/XiaoMigros/xmms-tools/archive/main.zip)

## Included plugins:

### Align ottavas
- Horizontally align 8va markings over accidentals and/or grace notes
- Applies to a selection or the whole score
- Requires MuseScore 4.6 or later

### Auto-Slur melismas
- Automatically add slurs over dashed lyrics
- Don't add slurs for ties
- Customiseable note limit to avoid long ties
- Requires MuseScore 3 or 4

### Fix to line
- Fix notes to specific lines
- Requires MuseScore 4.4 or later

### ReBeamer
- Automatically rebeam music
- Option to beam over rests
- Requires MuseScore 3 or 4

### Score duration
- Display the total score duration
- Option to save it to score properties
- Requires MuseScore 3 or 4

### Smart add slurs
- Automatically adds slurs to the selected passage of music, stopping at rests.
- Requires MuseScore 3 or 4

### Smart tie placement
- Switches all ties within chords to inside placement, all others to outside placement
- Requires MuseScore 4.6 or later

### Evenly Distribute Systems Across Pages
- Create balanced-looking sheet music with the help of this plugin
- Freely adjust the minimum or maximum number of systems per page
- Optional special rules for the first page
- Choose between various spacing density settings
- Requires MuseScore 4.6 or later

### Split Measure Across System
- Easily create a measure that spans over more than one system with the help of this plugin
- The plugin accounts for correct measure numbering, barline visibility, and the system of the measure itself
- Requires MuseScore 4.6 or later

### Tap BPM
- Select a BPM marking in your score, and tap the tempo
- The BPM marking will be adjusted automatically

## Installation

Download [all the files](https://github.com/XiaoMigros/xmms-tools/archive/main.zip), unzip them and move them to MuseScore's plugins folder.

For more help installing this plugin, visit [this page (v3)](https://musescore.org/en/handbook/3/plugins#installation) or [this page (v4)](https://musescore.org/en/handbook/4/plugins#install-new).

## Usage
Run the plugins from the 'Plugins' tab (in the 'System Spacing' subfolder in MU3). Except for split measures, the plugins apply to the whole score, not a selection.
